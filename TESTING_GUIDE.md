# 触屏功能测试指南

## 🎯 测试目标
验证坦克大战游戏的手机触屏功能是否正常工作

## 🧪 测试方法

### 方法一：浏览器开发者工具模拟（推荐）

1. **打开游戏页面**
   - 访问: http://localhost:8000
   - 点击"开始游戏"进入坦克大战

2. **启用设备模拟**
   - 按 F12 打开开发者工具
   - 点击左上角的"设备切换"图标📱（或按 Ctrl+Shift+M）
   - 选择移动设备如 iPhone 12 Pro 或 Pixel 5

3. **测试触屏功能**
   - 观察屏幕左下角是否出现蓝色虚拟摇杆
   - 观察屏幕右下角是否出现红色射击按钮
   - 用鼠标拖拽虚拟摇杆测试移动功能
   - 点击射击按钮测试开火功能

### 方法二：真实手机测试

1. **确保手机和电脑在同一网络**
2. **获取电脑IP地址**
   ```bash
   ipconfig  # Windows
   ifconfig  # Mac/Linux
   ```
3. **手机访问**：http://[你的IP地址]:8000

### 方法三：浏览器缩放测试

1. 缩小浏览器窗口到手机尺寸（宽度 < 768px）
2. 刷新页面
3. 验证触屏控制UI是否正确显示

## ✅ 预期测试结果

### PC端表现：
- ❌ 不显示虚拟摇杆和射击按钮
- ✅ 使用 WASD 和空格键正常控制

### 移动端表现：
- ✅ 左下角显示蓝色圆形虚拟摇杆
- ✅ 右下角显示红色圆形射击按钮
- ✅ 摇杆支持8方向控制
- ✅ 按钮点击后坦克开火
- ❌ 键盘控制被禁用

## 🔍 具体测试步骤

### 虚拟摇杆测试：
1. 鼠标点击并拖拽摇杆中心点
2. 向上拖拽 → 坦克向上移动
3. 向右拖拽 → 坦克向右移动
4. 斜向拖拽 → 坦克斜向移动
5. 松开鼠标 → 摇杆回到中心位置

1. 点击红色射击按钮
2. 观察坦克是否发射黄色子弹
3. 快速连点测试射击间隔

### 游戏完整性测试：
1. 使用触屏控制完成一整个关卡
2. 验证血量、分数、关卡显示正常
3. 测试游戏结束和重新开始功能

## 🐛 常见问题排查

### 问题1：触屏控制不显示
**解决方案：**
- 确认浏览器窗口宽度 < 768px
- 刷新页面
- 检查是否有JavaScript错误

### 问题2：摇杆无法拖动
**解决方案：**
- 确保在摇杆区域内点击开始拖拽
- 检查控制台是否有错误信息

### 问题3：射击按钮无反应
**解决方案：**
- 确认点击的是红色按钮区域
- 检查射击冷却时间（15帧间隔）
