# 杨云天的AI小站

欢迎来到杨云天的AI技术博客和游戏天地！

## 项目介绍

这是一个集技术分享和个人随想于一体的个人网站，展示了：
- 🚀 AI人工智能技术
- 🎮 精彩的在线游戏
- 💡 技术创新驱动

## 功能特色

### 🏠 主页
- 现代化的响应式设计
- AI技术主题展示
- 流畅的页面导航

### 🎯 游戏中心
目前包含以下游戏：
1. **坦克大战** - 经典的单人对战游戏
   - **电脑操作**：WASD控制移动，空格键射击
   - **手机操作**：左侧虚拟摇杆控制移动，右侧射击按钮开火
   - 击败所有敌人进入下一关
   - 血量系统和积分机制
   - **全平台支持**：完美适配PC和移动设备

### 📚 技术展示
- 个人技能标签展示
- 技术栈介绍
- 持续学习的理念

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **游戏开发**: Canvas API
- **设计**: 响应式布局, CSS Grid/Flexbox
- **特效**: CSS动画, 渐变背景
- **移动端**: 触屏API, 虚拟摇杆控制

## 运行方式

### 方法一：使用Python内置服务器
```bash
python -m http.server 8000
```

### 方法二：使用Node.js http-server
```bash
npx http-server -p 8000
```

### 方法三：直接打开文件
在浏览器中直接打开 `index.html` 文件

访问地址：http://localhost:8000

## 游戏操作说明

### 坦克大战
**电脑端操作：**
- **移动**: W(上) A(左) S(下) D(右)
- **射击**: 空格键
- **目标**: 击败所有敌方坦克

**移动端操作：**
- **移动**: 屏幕左下角虚拟摇杆
- **射击**: 屏幕右下角射击按钮
- **目标**: 击败所有敌方坦克

**游戏机制：**
- 玩家有100点生命值
- 每击败一个敌人获得100分
- 敌人会自动追踪并攻击玩家
- 完成一关后难度递增

## 特色功能

1. **响应式设计** - 适配各种设备尺寸
2. **现代化UI** - 渐变背景和毛玻璃效果
3. **流畅动画** - 悬停效果和过渡动画
4. **完整游戏体验** - 包含开始、暂停、重启功能
5. **跨平台支持** - 同时支持PC键盘和手机触屏操作
6. **性能优化** - 使用requestAnimationFrame优化渲染
7. **触屏优化** - 专为移动设备设计的虚拟控制界面
