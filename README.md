# 杨云天的AI小站

欢迎来到杨云天的AI技术博客和游戏天地！

## 项目介绍

这是一个集技术分享和个人随想于一体的个人网站，展示了：
- 🚀 AI人工智能技术
- 🎮 精彩的在线游戏
- 💡 技术创新驱动

## 功能特色

### 🏠 主页
- 现代化的响应式设计
- AI技术主题展示
- 流畅的页面导航

### 🎯 游戏中心
目前包含以下游戏：
1. **坦克大战** - 经典的单人对战游戏
   - WASD控制移动
   - 空格键射击
   - 击败所有敌人进入下一关
   - 血量系统和积分机制

### 📚 技术展示
- 个人技能标签展示
- 技术栈介绍
- 持续学习的理念

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **游戏开发**: Canvas API
- **设计**: 响应式布局, CSS Grid/Flexbox
- **特效**: CSS动画, 渐变背景

## 运行方式

### 方法一：使用Python内置服务器
```bash
python -m http.server 8000
```

### 方法二：使用Node.js http-server
```bash
npx http-server -p 8000
```

### 方法三：直接打开文件
在浏览器中直接打开 `index.html` 文件

访问地址：http://localhost:8000

## 游戏操作说明

### 坦克大战
- **移动**: W(上) A(左) S(下) D(右)
- **射击**: 空格键
- **目标**: 击败所有敌方坦克
- **机制**: 
  - 玩家有100点生命值
  - 每击败一个敌人获得100分
  - 敌人会自动追踪并攻击玩家
  - 完成一关后难度递增
