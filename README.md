# 杨云天的AI小站

欢迎来到杨云天的AI技术博客和游戏天地！

## 快速开始

### 🚀 运行项目
```bash
python -m http.server 8000
```
访问地址：http://localhost:8000

## 📚 详细文档

**坦克大战完整说明**：请查看 [坦克大战说明文档.md](坦克大战说明文档.md) 获取详细的游戏机制、操作指南和技术实现。

## 核心功能

### 🏠 主页特色
- 现代化的响应式设计
- AI技术主题展示
- 流畅的页面导航

### 🎯 游戏中心
**坦克大战** - 经典的单人对战游戏
- 第一关4个坦克，每关递增2个
- 左侧方向按钮控制移动，右侧射击按钮开火（移动端）
- 桌面端使用WASD移动，空格键射击
- 重新开始按钮始终居中显示

### 📱 移动端优化
- 响应式设计适配不同屏幕尺寸
- 触屏友好的虚拟按键布局
- 流畅的游戏体验

### 📚 技术展示
- 个人技能标签展示
- 技术栈详细介绍

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **游戏开发**: Canvas API
- **设计**: 响应式布局, CSS Grid/Flexbox
- **特效**: CSS动画, 渐变背景
- **移动端**: 触屏API, 虚拟按键控制
