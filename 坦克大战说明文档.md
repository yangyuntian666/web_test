## 🎮 坦克大战游戏详情

### 🎯 游戏机制
- **目标**：击败所有敌方坦克进入下一关
- **初始设置**：第一关4个坦克，每关递增2个
- **生命系统**：玩家初始100点生命值
- **积分机制**：每击败一个敌人获得100分
- **难度递增**：完成关卡后敌人数量和射击频率增加

### ⌨️ 操作方式

#### 电脑端操作
- **移动控制**：W(上) A(左) S(下) D(右)
- **射击控制**：空格键
- **游戏控制**：ESC键暂停，R键重新开始

#### 移动端操作
- **移动控制**：屏幕左下角的四个独立方向按钮（↑↓←→）
- **射击控制**：屏幕右下角独立射击按钮
- **重新开始**：底部居中重新开始按钮

### 📱 移动端优化特色

#### 🎮 控制优化
- **统一按钮尺寸**：四个方向键均为55×55px（小屏幕48×48px）
- **独立射击按钮**：65×65px（小屏幕55×55px），避免操作冲突
- **精准控制**：独立的方向键设计，避免误触
- **左右手分工**：方向键左下角，射击键右下角

#### 🖼️ 界面优化
- **游戏画布高度**：60vh（小屏幕55vh），合理比例不拉伸
- **按钮布局**：控制按钮位于屏幕边缘空白区域
- **不遮挡内容**：充分利用屏幕空间
- **安全区域适配**：兼容刘海屏等异形屏幕

#### 🚀 技术实现
- **响应式设计**：适配不同屏幕尺寸
- **触屏优化**：完整的触摸事件处理和视觉反馈
- **性能优化**：流畅的操作体验和快速响应
- **桌面端简洁**：隐藏虚拟按键，保持界面清爽

## 🛠️ 技术实现细节

### 🎮 游戏核心类

#### Tank类
```javascript
class Tank {
    constructor(x, y, color, isPlayer = false) {
        this.x = x;
        this.y = y;
        this.width = 40;
        this.height = 40;
        this.color = color;
        this.speed = isPlayer ? 3 : 1;
        this.direction = 0; // 0:上, 1:右, 2:下, 3:左
        this.isPlayer = isPlayer;
        this.health = isPlayer ? 100 : 30;
        this.shootCooldown = 0;
        this.maxShootCooldown = isPlayer ? 15 : 60;
    }
}
```

#### Bullet类
```javascript
class Bullet {
    constructor(x, y, dirX, dirY, isPlayerBullet) {
        this.x = x;
        this.y = y;
        this.dirX = dirX;
        this.dirY = dirY;
        this.radius = 4;
        this.speed = 7;
        this.isPlayerBullet = isPlayerBullet;
    }
}
```

```javascript
function createVirtualButtons() {
    // 创建四个统一尺寸的方向按钮
    const directionArea = document.createElement('div');
    directionArea.innerHTML = `
        <button class="control-btn up-btn" id="upBtn">↑</button>
        <div class="horizontal-controls">
            <button class="control-btn left-btn" id="leftBtn">←</button>
            <button class="control-btn right-btn" id="rightBtn">→</button>
        </div>
        <button class="control-btn down-btn" id="downBtn">↓</button>
    `;
}
```
